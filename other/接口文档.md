# 接口文档
## 全局
```json
{
  "status":200,//一个三位整数
  "message":"OK",//一个简短的英文短语用于说明
  "data":{}//请求的数据
}
```

host    :   主机名  
url     :   后面拼接内容

## 返回信息格式的具体说明
```json
{
//代表成功返回
  "status":200,
  "message":"OK",
//
//代表失败，默认 failure
  "status":300,
  "message":"OK",
//
//代表没有权限
  "status":400,
  "message":"permission denied"
}
```
## 具体请求
### 无权限请求
#### 登录  
url:/login  
method:POST  
request
```json
{"username":"id","password":"password"}
```
response
```json
{
  "status":200,
  "message":"OK",
  "data":{}
}
```
```json
{
  "status":300,//300包括密码错误等
  "message":"用户名或密码错误",
  "data":{}
}
```

url:/register  
method:POST  
request
```json
{

  "type":"student"//或者"teacher"

```
response
```json

```
